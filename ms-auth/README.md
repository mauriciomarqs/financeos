# ms-auth

Microservico de autenticacao do FinanceOS. Login, JWT e perfis de usuario.

**Porta:** 8082

## Endpoints

| Metodo | Endpoint | Descricao |
|--------|----------|-----------|
| POST | /auth/register | Cadastrar usuario |
| POST | /auth/login | Login e retorno do JWT |
| GET | /auth/me | Dados do usuario logado |
